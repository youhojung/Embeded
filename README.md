## 기본 기능  
- 빨간 LED (2초) → 노란 LED (0.5초) → 초록 LED (2초) → 초록 LED 깜빡임 (3회, 총 1초) 반복  

## 모드 전환  
버튼을 누르면 **Emergency, Blink All, On Off** 모드로 전환할 수 있다.  
다시 한 번 버튼을 누르면 **Normal 모드**로 돌아가며, 빨간 LED가 켜지는 것부터 다시 실행된다.
